Endpoints

Can either use Postman or the browser to view the different endpoints
Note these links assume you're running on port 3010

Totals

Carriers Count (http://localhost:3010/api/v1/carriers/total_count)
Client Count (http://localhost:3010/api/v1/clients/total_count)
Policies Count (http://localhost:3010/api/v1/policies/total_count)

Pagenated Client List

Client Listing (http://localhost:3010/api/v1/clients/list?page=5)
